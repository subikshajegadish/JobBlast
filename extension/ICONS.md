# Extension Icons

The extension requires three icon files in the `icons/` directory:

- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

## Creating Icons

You can create these icons using:

1. **Online Tools**:
   - [Favicon Generator](https://www.favicon-generator.org/)
   - [RealFaviconGenerator](https://realfavicongenerator.net/)
   - [Favicon.io](https://favicon.io/)

2. **Image Editors**:
   - GIMP
   - Photoshop
   - Figma
   - Canva

3. **Quick Solution**:
   Create a simple icon with text "JAT" (Job Application Tracker) or use any job-related icon.

## Icon Guidelines

- Use a simple, recognizable design
- Ensure icons are clear at small sizes (16x16)
- Use a transparent background (PNG format)
- Consider using a briefcase, document, or checkmark icon

## Placeholder Icons

For development, you can use any 16x16, 48x48, and 128x128 PNG images. The extension will work without icons, but Chrome will show a default icon.
